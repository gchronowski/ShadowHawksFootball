How to Use This Site

• Roster: View player positions and numbers.
• Injury Report: Check current player availability.
• Standings: See division rankings and playoff chances.
• Home Page: Team updates and announcements.

Editing This Website

If you're updating the site manually:

  • Open the page you want to edit (HTML file).
  • Look for the comment blocks indicating where information should be updated.
  • Save your changes and commit them to the repo.

Contact

For questions, reach out to the site administrator or coaching staff.
